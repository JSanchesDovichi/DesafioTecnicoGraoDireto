# Como debugar

Dependências:

- Docker

```sh
docker compose up -D
```

- Google Chrome / Chromium

```sh
Se estiver usando o Chromium, use a opção correspondente no VSCode.
```

- Rust

```sh
cd back_end &&
cargo run
```

- Flutter

```sh
  Use a opção do VSCode para o navegador correspondente.
```
